# Contrato Felesa CRONO hidraulico

Contrato independiente para el formulario `FORMULARIO MANIOBRA CRONO HIDRAULICAS`.

- `profile_id`: `felesa_crono`
- `template_id`: `felesa_crono_hidraulico`
- `form_version`: `0`
- Alcance inicial: solo pagina tecnica principal.

Este contrato no comparte secciones con el contrato electrico. Las reglas que coincidan se podran copiar, pero cada campo queda definido en este directorio.
